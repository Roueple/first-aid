# Department Categories Reference

## Overview

This document lists all department categories with their keywords, common abbreviations, and real-world examples from the audit results database.

## Categories

### 1. IT (Information Technology)
**Keywords:** it, teknologi, informasi, technology, ict, sistem informasi

**Common Names:**
- Departemen IT
- Departemen Teknologi Informasi
- ICT Department
- Information & Communication Technology
- Audit IT - Asset Teknologi Informasi

- Audit IT - General Control
- Manajemen Risiko Teknologi Informasi dan Keamanan Informasi

## Maintenance

When adding new department names:
1. Check if existing keywords match
2. If not, consider adding new keywords to appropriate category
3. If completely new domain, consider creating new category
4. Update this reference document
