# Product Overview

FIRST-AID is an Electron-based desktop application for managing audit findings in Indonesian real estate projects. It combines traditional database management with AI-powered natural language querying.

## Core Capabilities

- **Audit Results Management**: Track 8,840+ audit findings across 110+ real estate projects (landed houses, apartments, hotels, hospitals, clinics, schools, universities)
- **AI Chat Assistant**: Natural language queries powered by Google Gemini AI with Indonesian real estate domain knowledge (200+ terms: PPJB, AJB, SHM, IMB, etc.)
- **Smart Query Routing**: Automatically routes queries to SQL, semantic search (RAG), or hybrid approaches based on intent
- **Data Privacy**: Automatic pseudonymization and masking of sensitive data during AI processing
- **Excel Integration**: Import/export capabilities for projects and audit results
- **Real-time Collaboration**: Firebase-powered real-time updates across users

## Key Features

- Hybrid RAG system with semantic search (85% accuracy) and keyword fallback
- Smart filter extraction from natural language (98% accuracy in hybrid mode)
- Indonesian real estate terminology understanding (PPJB, AJB, SHM, IMB, KPR, etc.)
- Project management with unique 7-digit IDs and 3-4 character initials
- Audit results with SHA-256 unique IDs and duplicate detection
- Chat history with session management and automatic title generation
- Department normalization and categorization

## Target Users

Internal audit teams at Indonesian real estate companies who need to:
- Query audit findings using natural language
- Track findings across multiple projects and years
- Generate reports and export data
- Understand patterns and trends in audit results
